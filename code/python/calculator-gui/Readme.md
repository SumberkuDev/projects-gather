# Python Calculator Gui 
# How to Use 

- Clone Repository 
    ```bash
    git clone https://github.com/anandadwipra/projects-gather.git
    ```
- Open Folder Location
    ```bash
    cd projects-gather/code/python/calculator-gui
- Run Program
    ```bash
    python main.py 
    #or
    python3 main.py
    ```
Please Make sure you use python3
